# Week 6 - Image Denoising Using Autoencoder

## Objective
Build a Deep Learning model to remove noise from MNIST handwritten digit images using a Convolutional Autoencoder.

## Topics Covered
- Autoencoder
- Variational Autoencoder (VAE)
- GAN (Generator & Discriminator)
- SGD with Momentum
- AdaGrad
- RMSprop
- Adam Optimizer
- Introduction to Generative AI
- RAG (LangChain)
- Prompt Engineering

## Tools Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- VS Code

## Files
- `autoencoder.py` – Source code
- `noisy_images.png` – Noisy input images
- `denoised_results.png` – Original, noisy, and denoised images
- `Report.docx` – Project report
- `mnist_autoencoder.keras` – Trained model

## Steps
1. Load and preprocess the MNIST dataset.
2. Add Gaussian noise to the images.
3. Build and train a Convolutional Autoencoder.
4. Generate denoised images.
5. Compare original, noisy, and reconstructed images.

## Output
- Successfully removed noise from MNIST images.
- Reconstructed images are similar to the original images.
