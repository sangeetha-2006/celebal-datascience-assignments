# Week 3 - Customer Intelligence System

## Assignment

Develop a Customer Intelligence System using classification, ensemble learning, and clustering techniques.

## Dataset

* Country-data.csv
* data-dictionary.csv

## Techniques Used

* Data Cleaning
* StandardScaler
* K-Means Clustering
* DBSCAN Clustering
* PCA
* Silhouette Score

## Results

* Elbow Method used for cluster selection
* K-Means model trained with K=3
* DBSCAN comparison model created
* PCA visualization generated
* Socio-economic observations documented
